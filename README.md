# 炸彈人遊戲

這遊戲原本是以前在用VB製作專題時寫的遊戲，現在用網頁復刻，也改良了一些寫法和bug，並支援電腦和部分手機版。

## 規則

可以炸毀所有金塊以分數取勝，或殺死對方取勝。
